## Think.js
Version 2.0

### Perceptron

```js
let p = Think.perceptron([0, 1], 1);
let res = p.process(); // 0.972...
```

### K-Means
```js
let data =[[162,41],[197,145]....];
let means = Think.means("canvas", data, 3);
means.cluster();
```


### Linear regression
```js
let data =[[162,41],[197,145]....];
let r = Think.regression("canvas", data);
r.plot();
```
